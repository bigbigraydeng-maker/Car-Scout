# 🚀 方案2执行清单 - Render自动化配置

## 🎯 目标
让Render服务器自动抓取车辆数据，每天更新网站。

---

## ✅ 立即执行（5分钟见效）

### 步骤1：在Render Shell中手动运行一次

1. 打开 https://dashboard.render.com
2. 点击 **car-scout**
3. 点击 **Shell** 标签
4. **逐行执行**以下命令：

```bash
# 进入src目录
cd src

# 运行TradeMe抓取器（不需要登录）
node trademe-scraper.js

# 等待完成（约3-5分钟）...
# 看到"抓取完成"提示后，继续：

# 合并数据到数据库
node merge-trademe.js

# 筛选超里程车辆
node filter-by-mileage.js

# 生成看板
node dashboard.js dashboard

# 完成！
```

5. **刷新网站**：https://car-scout.onrender.com/
6. 应该能看到 **20+辆车** 了！

---

## 🕐 配置定时自动抓取（可选）

### 步骤2：创建定时任务

**在Render Dashboard中：**

1. 点击 **New +** → **Cron Job**
2. 填写配置：
   - **Name**: `car-scout-scraper`
   - **Runtime**: Node
   - **Build Command**: `npm install`
   - **Start Command**: `node src/auto-scrape.js`
   - **Schedule**: `0 8 * * *`（每天上午8点）
   - **Plan**: Free
3. 点击 **Create Cron Job**

**这样就完全自动化了！** ✅
