# Facebook Marketplace 站内消息模板
# Car Scout - 倒卖专用

## 模板 1: 初次询问 (标准版)

**适用场景**: 第一次联系卖家，询问基本情况

```
Hi there,

I'm interested in your Toyota [车型] listed for $[价格]. 

Could you please let me know:
1. What year is the car?
2. How many kilometers has it done?
3. When does the WOF expire?
4. Any mechanical issues or recent repairs?
5. Do you have service records?

I'm looking to buy this week and can pay cash. Would it be possible to view the car this weekend?

Thanks!
[你的名字]
```

**中文版本:**
```
你好，

我对你的Toyota [车型] (要价$[价格])很感兴趣。

请问能否告诉我：
1. 车的年份？
2. 行驶了多少公里？
3. WOF什么时候到期？
4. 有没有机械问题或最近维修过？
5. 有保养记录吗？

我这周就想买，可以现金支付。这周末可以安排看车吗？

谢谢！
[你的名字]
```

## 模板 2: 初次询问 (简短版)

**适用场景**: 快速筛选，确认基本信息

```
Hi,

Interested in your [车型]. Can you tell me:
- Year and mileage?
- WOF/Reg expiry?
- Any issues?

Available to view tomorrow? Cash buyer.

Thanks
```

## 模板 3: 砍价版 (看过车后)

**适用场景**: 看完车后发现一些小问题，用来砍价

```
Hi [卖家名字],

Thanks for showing me the car today. I noticed a few things:
- [问题1, e.g., "The tires look worn and will need replacing soon"]
- [问题2, e.g., "There's a small scratch on the rear bumper"]
- [问题3, e.g., "The interior needs a good clean"]

Given these items, would you consider $[你的出价] cash? I can pick it up this weekend.

Let me know!

Thanks,
[你的名字]
```

**中文版本:**
```
你好 [卖家名字]，

今天谢谢你给我看车。我注意到几个问题：
- [问题1，比如"轮胎磨损严重，需要更换"]
- [问题2，比如"后保险杠有小刮痕"]
- [问题3，比如"内饰需要清洁"]

考虑到这些，$[你的出价]现金可以吗？这周末可以提车。

等你回复！

谢谢，
[你的名字]
```

## 模板 4: 预约看车版

**适用场景**: 已经问完基本信息，约时间看车

```
Hi,

Thanks for the info. The car sounds good.

I'm available to view:
- Saturday morning (9am-12pm)
- Sunday afternoon (2pm-5pm)

Which works better for you? I'll bring cash and can buy on the spot if everything checks out.

Cheers
```

**中文版本:**
```
你好，

谢谢提供信息。车听起来不错。

我可以看车的时间：
- 周六上午 (9点-12点)
- 周日下午 (2点-5点)

哪个时间对你方便？我会带现金，车况没问题的话可以当场买。

谢谢
```

## 模板 5: 确认购买版

**适用场景**: 砍价成功，确认交易细节

```
Hi [卖家名字],

Great! $[成交价] cash works for me.

I'll come to pick up the car on [日期] at [时间]. 

Please bring:
- Keys
- Registration papers
- Any service records

I'll bring the cash and a sale agreement to sign.

See you then!

[你的名字]
```

**中文版本:**
```
你好 [卖家名字]，

太好了！$[成交价]现金没问题。

我[日期] [时间]来提车。

请准备好：
- 车钥匙
- 车辆登记文件
- 保养记录（如果有）

到时候见！

```

## 使用建议

### 发送顺序
1. **第一轮**: 使用模板1或2，同时发给TOP 5的卖家
2. **等回复**: 24小时内回复率通常在60-80%
3. **筛选**: 根据回复筛选出2-3辆最有兴趣的
4. **预约**: 使用模板4约时间看车
5. **砍价**: 看完车后用模板3砍价
6. **成交**: 谈妥后用模板5确认交易

### 注意事项
- 使用英文发送（新西兰主流）
- 回复要快（30分钟内最佳）
- 保持礼貌但直接
- 强调"cash buyer"增加吸引力
- 不要透露你是倒卖，说"个人使用"

### 快速回复技巧
如果卖家回复慢，可以发跟进消息：
```
Hi, just following up on my message. Are you still selling the car?

Thanks
```

## 常见问题回复

**卖家问: "What's your best offer?"**
回复: 
```
I need to see the car first to assess the condition. Can we arrange a viewing?
```

**卖家问: "Why are you buying?"**
回复:
```
Looking for a reliable daily driver. Heard good things about Toyota reliability.
```

**卖家说: "Someone else is interested"**
回复:
```
I understand. I'm a serious cash buyer and can commit today if the car is as described. Let me know if the other deal falls through.
```
