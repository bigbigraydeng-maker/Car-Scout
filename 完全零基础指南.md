## 🎯 每天只需3步

### 第1步：抓取车辆（早上）
1. **双击** `1-开始抓取.bat`
2. 等待浏览器打开
3. 如果提示登录Facebook，输入账号密码
4. 等待完成（约10分钟）
5. 按任意键关闭窗口

### 第2步：生成报告
1. **双击** `2-生成报告.bat`
2. 等待几秒
3. 自动打开报告文件夹

### 第3步：查看车辆
1. **双击** `打开车辆看板.bat`
2. 浏览器自动打开
3. 查看所有车辆信息

---

## 🖥️ 车辆管理看板怎么用？

### 打开看板后，你会看到：

**1. 顶部统计**
```
8辆车  |  1辆新  |  8辆活跃  |  4项待办
```

**2. 今日待办清单**
- 🔴 红色 = 高优先级（必须今天做）
- 🟡 黄色 = 中优先级（可以等等）
- 点击"完成"标记已做完
- 点击"看车清单"生成检查表

**3. 工作流程图**
```
📥1 → 📞1 → 📅1 → 🔍1 → 💰1 → ✅1 → 🧹1 → 🔄1
```
显示每个阶段有几辆车

**4. 筛选标签**
- 点击标签筛选车辆
- 例如：点击"🆕 新发现"只看新车

**5. 车辆卡片**
每张卡片显示：
- 🚗 年份和车型
- 💰 价格
- 🛣️ 里程
- ⭐ 优先级（1-9分，越高越好）
- 🏷️ 当前状态
- 🔗 点击"查看"打开Facebook
- ➡️ 点击"推进"到下一阶段

## 📝 看车流程

### 1. 预约看车前
1. 打开车辆看板
2. 找到要看的车辆
3. **双击** `5-生成看车清单.bat`
4. 输入车辆ID（从卡片上复制）
5. 自动生成检查清单
6. 打印或用手机查看

### 2. 看车时
- 按清单逐项检查
- 记录问题和车况
- 拍照留档

### 3. 看车后
1. 回到看板
2. 点击车辆的"推进"按钮
3. 选择下一阶段（例如：车检中）

---

## 🔄 如何推进车辆状态？

在看板中：
1. 找到车辆卡片
2. 点击"➡️ 推进"按钮
3. 确认推进
4. 车辆自动进入下一阶段

### 各阶段说明：
```
📥 新发现 → 📞 已联系 → 📅 预约看车 → 🔍 车检中 → 
💰 议价中 → ✅ 已购买 → 🧹 清理中 → 🔄 已上架 → 🏁 已售出
```

---

## 📊 如何理解优先级？

优先级1-9分：
- **7-9分**（红色）：极好，立即联系
- **4-6分**（黄色）：不错，可以考虑
- **1-3分**（绿色）：一般，可以等等

优先级根据价格、里程、年份自动计算

## ❓ 常见问题

### Q: 双击没反应？
**A:** 
1. 先双击 `0-安装依赖.bat`
2. 如果还不行，可能需要安装Node.js
3. 查看"安装Node.js指南.md"

### Q: 为什么每次都要登录Facebook？
**A:** 
- 首次需要登录
- 之后会自动记住
- 如果提示过期，重新登录一次

### Q: 数据会丢失吗？
**A:** 不会！所有数据自动保存在 `database/` 文件夹中

### Q: 如何备份数据？
**A:** 复制 `database/vehicles.json` 到其他位置即可

### Q: 可以同时管理几辆车？
**A:** 可以同时管理几十辆车，看板会显示所有车辆

### Q: 价格变动会提醒吗？
**A:** 会！下次抓取时如果价格变了，会在待办中提醒

## 💡 使用技巧

### 技巧1：每天早上
1. 运行 `1-开始抓取.bat`
2. 运行 `2-生成报告.bat`
3. 打开看板查看"今日待办"

### 技巧2：按优先级处理
- 先看红色标记的高优先级车辆
- 中低优先级可以稍后处理

### 技巧3：使用筛选
- 点击筛选标签快速找到特定阶段的车辆
- 例如：只看"待看车"或"车检中"

### 技巧4：打印看车清单
- 看车前一定要生成清单
- 打印出来或存在手机里
- 逐项检查不遗漏

- 在看板页面点击"🔄 刷新数据"按钮
