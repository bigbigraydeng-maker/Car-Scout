# 🚗 Car Scout - 快速参考卡

## 每天只需3步：

### 1️⃣ 抓取车辆 → 双击：1-开始抓取.bat
- 浏览器自动打开
- 登录Facebook（首次）
- 等待10分钟
- 自动保存数据

### 2️⃣ 生成报告 → 双击：2-生成报告.bat  
- 自动生成3个文件
- 自动打开文件夹

### 3️⃣ 查看报告 → 双击：4-打开看板.bat
- 双击最新的HTML文件
- 浏览器中查看

---

## 其他功能：

### 查看今天做什么 → 双击：3-查看待办.bat

### 准备看车 → 双击：5-生成看车清单.bat
- 输入车辆ID
- 自动生成清单

### 第一次使用 → 双击：0-安装依赖.bat

## 🎯 优先级颜色：

- 🔴 红色 = 极高优先级（立即联系）
- 🟠 橙色 = 高优先级（今天处理）
- 🟢 绿色 = 低优先级（可以等等）
