## ⚠️ 重要提醒

### 免费版限制
- **休眠机制**：15分钟无访问后休眠
- **唤醒时间**：下次访问需等待10-30秒
- **解决**：每天早上自己访问一次保持活跃

### 数据备份
- **数据库位置**：`database/vehicles.json`
- **建议**：每周下载备份一次
- **原因**：Render可能偶尔重置文件系统
