## 📋 项目简介

**Car Scout** 是一个自动化 Toyota 二手车扫描系统，每天扫描三个平台寻找符合投资标准的车辆。

### 支持平台
1. **Facebook Marketplace** - 私人卖家，可议价
2. **TradeMe** - 新西兰最大二手车平台
3. **Skykiwi 论坛** - 华人社区，中文沟通

---

## 🎯 核心功能

- ✅ **每日自动扫描** - 每天早上 8:00 自动执行
- ✅ **智能筛选** - 自动过滤不符合条件的车辆
- ✅ **评分排名** - 100分制评分系统，找出最佳机会
- ✅ **飞书通知** - 高评分车辆即时推送
- ✅ **三平台汇总** - 一个报告看遍所有平台

---

## 📊 筛选标准

| 条件 | 值 |
|------|-----|
| **价格** | $2,000 - $5,000 NZD |
| **年份** | 2005年及以上 |
| **里程** | ≤ 200,000 km |
| **车型** | Corolla, Vitz, Wish, RAV4 |
| **地区** | Auckland, Waikato |

---

## ⭐ 评分维度

| 维度 | 权重 |
|------|------|
| 价格优势 | 25分 |
| 车况/里程 | 20分 |
| 车型热度 | 15分 |
| 年份折旧 | 15分 |
| 地区便利性 | 10分 |
| 描述质量 | 10分 |
| 周转速度 | 5分 |

**风险等级**: 🟢低风险(80+) | 🟡中风险(60-79) | 🔴高风险(<60)
