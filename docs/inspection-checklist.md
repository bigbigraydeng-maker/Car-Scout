# 🚗 二手车看车检查清单

## 📋 基本信息核对

### 证件检查
- [ ] **WOF (车辆安全检测)** - 检查有效期
- [ ] **Rego (路税)** - 检查有效期
- [ ] **车辆登记证** - 核对车主姓名
- [ ] **保养记录** - 最后一次保养时间和内容

### 车辆识别
- [ ] 核对车牌号与广告一致
- [ ] 核对车架号 (VIN)
- [ ] 确认年份、型号无误

---

## 🔧 外观检查

### 车身
- [ ] 漆面 - 是否有重新喷漆痕迹
- [ ] 锈迹 - 重点检查：
  - 轮拱内侧
  - 车门底部边缘
  - 底盘
  - 引擎盖边缘
- [ ] 面板缝隙 - 是否均匀（事故修复迹象）
- [ ] 玻璃 - 是否有裂纹、更换痕迹

### 2004 Vitz 特别注意 ⚠️
- [ ] 后保险杠是否错位（卖家已说明）
- [ ] 右前轮附近刮痕（卖家已说明）
- [ ] 表面锈迹程度

---

## 🚗 内饰检查

- [ ] 座椅磨损程度
- [ ] 地毯是否潮湿（漏水迹象）
- [ ] 仪表盘所有指示灯是否正常
- [ ] 空调制冷效果
- [ ] 电动车窗、门锁
- [ ] 音响系统
- [ ] 内饰异味（烟味、霉味）

---

## ⚙️ 引擎舱检查

### 冷启动（关键！）
- [ ] 冷车启动是否顺畅
- [ ] 启动时有无异响
- [ ] 怠速是否稳定（约700-900 RPM）

### 运转检查
- [ ] 引擎声音是否平稳
- [ ] 有无敲击声、咔哒声
- [ ] 排气颜色（正常应为无色或轻微白色）
  - ⚠️ 蓝烟 = 烧机油
  - ⚠️ 黑烟 = 燃烧不充分

### 液体检查
- [ ] 机油颜色和液位（应在标尺中间）
- [ ] 冷却液液位和颜色
- [ ] 刹车油液位
- [ ] 有无漏油痕迹（引擎底部、地面）

---

## 🛣️ 试驾检查（最重要）

### 出发前
- [ ] 方向盘有无虚位
- [ ] 刹车踏板感觉（不应过软或过硬）

### 行驶中
- [ ] **变速箱换挡** - 是否平顺，无顿挫
- [ ] **加速性能** - 油门响应是否正常
- [ ] **转向手感** - 是否偏重或有异响
- [ ] **直线行驶** - 松手后是否跑偏

### 特定速度测试
- [ ] 20-40 km/h：低速换挡
- [ ] 50-60 km/h：巡航稳定性
- [ ] 80-100 km/h：高速稳定性、风噪

### 刹车测试（安全路段）
- [ ] 40 km/h 中等力度刹车 - 是否跑偏
- [ ] 刹车是否有异响
- [ ] 刹车盘是否抖动

### 其他
- [ ] 悬挂过减速带感觉
- [ ] 车内是否有异响
- [ ] 风噪、路噪是否正常

---

## 🛞 底盘和轮胎

- [ ] 轮胎品牌和生产日期（4位数字，如"2518"=2018年第25周）
- [ ] 轮胎磨损程度（花纹深度）
- [ ] 四轮磨损是否均匀
- [ ] 备胎及工具是否齐全
- [ ] 底盘有无明显锈蚀或损坏

## 🎯 针对 2004 Vitz $3,900 的评估重点

### 关键决策点
1. **机械状况** - 引擎和变速箱是否正常
2. **锈蚀程度** - 是否影响安全和寿命
3. **整修成本** - 保险杠+刮痕+清洁 预估$600-900
4. **最终利润** - 按计算器，建议售价$5,060，利润$960

### 建议策略
- **能压价到$3,200**：强烈推荐购买
- **$3,400-$3,600**：可以考虑
- **高于$3,600**：利润空间太小，建议放弃
