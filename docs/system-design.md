# 🚗 Car Scout 内部管理系统

## 项目概述
开发一个内部管理后台，用于：
- 车辆数据管理
- 跟进状态跟踪
- 利润计算
- 自动发布到论坛（小红书、Skykiwi）

## 技术方案

### 方案一：简单 Web 应用（推荐）
- **前端**: HTML + CSS + JavaScript
- **后端**: Node.js + Express
- **数据库**: JSON 文件（轻量级）
- **部署**: 本地运行，浏览器访问

### 方案二：桌面应用
- **技术**: Electron
- **优点**: 独立窗口，更像原生应用
- **缺点**: 体积较大

## 推荐：方案一 - Web 应用

### 功能模块

#### 1. 仪表盘 📊
- 今日新增车辆
- 待跟进车辆
- 利润统计
- 最近活动

#### 2. 车辆管理 🚗
- 车辆列表（表格视图）
- 添加/编辑/删除车辆
- 筛选和搜索
- 排序功能

#### 3. 利润计算器 💰
- 输入：收购价、年份、里程、车况
- 输出：建议售价、预计利润、风险评级
- 可视化图表

#### 4. 跟进管理 📋
- 跟进状态（新发现/已联系/已看车/谈判中/已购买/已出售）
- 备注和日志
- 提醒功能

#### 5. 自动发布 🤖
- 一键生成帖子内容
- 发布到 Skykiwi
- 发布到小红书（需要API）
