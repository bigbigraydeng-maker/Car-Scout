# 🌐 Car Scout - 在线共享版部署指南

## 目标：让合伙人可以在任何地方查看车辆信息

---

## 方案1：本地Web服务器（最简单，适合局域网）

### 适合场景
- 你和合伙人在同一个办公室/家里
- 或者通过VPN连接

### 使用方法

1. **双击启动**
   ```
   启动Web服务器.bat
   ```

2. **查看地址**
   - 本机访问：`http://localhost:3000`
   - 局域网访问：`http://[你的IP]:3000`
   
3. **分享给合伙人**
   - 告诉合伙人你的IP地址
   - 例如：`http://192.168.1.100:3000`

---

## 方案2：部署到Render（推荐，免费在线访问）

### 适合场景
- 合伙人分布在不同地区
- 需要24小时在线访问
- 完全免费

### 步骤

#### 第1步：注册Render账号
1. 打开 https://render.com
2. 用GitHub账号登录
3. 点击 "New Web Service"

#### 第2步：上传代码
1. 将整个Car Scout文件夹上传到GitHub
2. 在Render中选择这个GitHub仓库
3. 配置如下：

```
Name: car-scout
Runtime: Node
Build Command: npm install
Start Command: node src/web-server.js
Port: 3000
```

#### 第3步：获取访问地址
- Render会给你分配一个免费域名
- 例如：`https://car-scout.onrender.com`
- 将这个地址分享给合伙人

#### 优点
- ✅ 完全免费
- ✅ 24小时在线
- ✅ 自动HTTPS
- ✅ 合伙人随时访问

## 方案3：部署到Vercel（最简单，推荐）

### 适合场景
- 不想配置服务器
- 一键部署
- 全球CDN加速

### 步骤

#### 第1步：准备代码
1. 创建 `vercel.json` 文件：

```json
{
  "version": 2,
  "builds": [
    {
      "src": "src/web-server.js",
      "use": "@vercel/node"
    }
  ],
  "routes": [
    {
      "src": "/(.*)",
      "dest": "src/web-server.js"
    }
  ]
}
```

#### 第2步：部署
1. 打开 https://vercel.com
2. 用GitHub登录
3. 导入Car Scout仓库
4. 点击 Deploy

#### 第3步：分享
- 自动获得域名：`https://car-scout-yourname.vercel.app`
- 直接分享给合伙人

## 方案4：使用内网穿透（ngrok）

### 适合场景
- 临时分享给远程合伙人
- 不想部署到服务器
- 5分钟搞定

### 步骤

#### 第1步：安装ngrok
1. 下载：https://ngrok.com/download
2. 解压到Car Scout文件夹

#### 第2步：启动本地服务器
```
启动Web服务器.bat
```

#### 第3步：启动ngrok
```
ngrok http 3000
```

#### 第4步：获取公网地址
- ngrok会显示：`https://xxxx.ngrok.io`
- 将这个地址分享给合伙人
- 地址24小时有效

## 🔒 安全建议

### 1. 设置访问密码（可选）

修改 `src/web-server.js`，添加密码验证：

```javascript
// 在generateHTML之前添加验证
const auth = req.headers.authorization;
if (!auth || auth !== 'Bearer your-secret-key') {
  res.statusCode = 401;
  res.end('Unauthorized');
  return;
}
```

### 2. 使用HTTPS
- Render和Vercel自动提供HTTPS
- ngrok也提供HTTPS

### 3. 定期更换地址
- 如果使用ngrok，地址会变化
- 如果使用Render/Vercel，地址固定

## 📞 常见问题

### Q: 合伙人打不开网页？
**A:** 
1. 检查你的服务器是否运行中
2. 检查防火墙是否允许端口
3. 如果使用ngrok，确认地址未过期

### Q: 数据会实时同步吗？
**A:** 
- 页面每30秒自动刷新
- 合伙人看到最新数据

### Q: 合伙人可以修改数据吗？
**A:**
- 当前版本只支持查看
- 修改需要在你的电脑上操作

### Q: 可以同时多人访问吗？
**A:**
- 可以，没有人数限制
- 所有人都看到相同数据
